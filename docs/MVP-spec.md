# 图片背景移除工具 · MVP 需求文档

> 版本：v1.0.0 · 2026-03-28

---

## 1. 产品概述

**产品名称：** bg-remover（待定）

**一句话描述：** 拖拽上传图片，自动移除背景，支持 PNG 透明通道下载。

**核心价值：** 最简单的在线背景移除工具，无需注册、无需等待、即用即走。

**技术选型：**
- 前端：原生 HTML/CSS/JS（无框架，单文件交付）
- 后端：Cloudflare Workers（边缘计算，全球低延迟）
- AI 能力：Remove.bg API（成熟方案，快速上线）
- 存储：**无**（纯内存处理，不留存任何数据）

---

## 2. 用户故事

| 角色 | 场景 | 需求 |
|------|------|------|
| 普通用户 | 想要给证件照换背景色 | 上传 → 自动移除背景 → 下载 PNG |
| 设计师 | 快速提取产品图主体 | 拖拽上传 → 秒级出图 → 下载使用 |
| 运营 | 需要透明底素材做海报 | 上传商品图 → 去背景 → 导出 |

---

## 3. 功能范围

### 3.1 必做（MVP）

**上传**
- 支持拖拽上传 + 点击选择文件
- 支持格式：JPG / PNG / WebP
- 文件大小限制：10MB（Remove.bg API 上限）
- 前端校验文件类型和大小，给予明确错误提示

**处理**
- 调用 Remove.bg API 进行背景移除
- 处理中显示加载状态（spinner + 文字提示）
- 错误时显示具体错误信息

**预览 & 下载**
- 同时展示原图和处理后图（左右对比）
- 一键下载结果为 PNG（透明背景）
- 不提供任何在线存储/历史记录功能

**页面**
- 纯前端单页，无需登录
- 移动端适配（响应式布局）
- 页面底部标注"图片仅在内存中处理，不做任何存储"

### 3.2 不做（本期）

- 用户账号体系
- 批量处理
- 历史记录 / 最近处理
- 付费订阅 / Stripe 接入
- API 对外开放
- 自托管模型

## 5. 页面设计规范

### 视觉风格
- 配色：深色主题（#0f0f0f 背景），减少视觉干扰，突出图片内容
- 字体：系统字体栈（-apple-system / Segoe UI），无需额外加载
- 圆角：16px 大圆角，友好现代

### 布局
- 单页居中布局，最大宽度 640px
- 拖拽区：虚线边框，吸引注意力
- 预览区：原图 / 结果左右并排，移动端改为上下排列

### 加载状态
- 中心 spinner + "正在移除背景…" 文字
- 全程不可二次提交（防止重复调用 API 浪费配额）

### 错误提示
- 非侵入式：仅在拖拽区下方显示红色错误文案
- 错误信息清晰（如"文件大小不能超过 10MB"而非"Error 413"）

## 6. API 设计

本项目**无自建 API**，Remove.bg API 调用在后端 Worker 内完成。

### Worker 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/` | 返回 HTML 上传页面 |
| POST | `/` | 接收 `image` 表单字段，返回 PNG 结果 |

### POST / （内部代理）

**请求：**
- Content-Type: `multipart/form-data`
- 字段：`image`（File）

**响应：**
- 成功：`Content-Type: image/png`，直接返回 PNG 二进制
- 失败：`Content-Type: text/plain`，返回错误信息 + 对应 HTTP 状态码

**错误码映射：**
| 情况 | 返回状态码 |
|------|-----------|
| 未提供图片 | 400 |
| Remove.bg API Key 未配置 | 500 |
| Remove.bg 返回错误 | 透传其状态码 |
| 处理异常 | 500 |

## 8. 限制与边界

| 维度 | 限制值 | 说明 |
|------|--------|------|
| 单文件大小 | ≤ 10MB | Remove.bg API 上限 |
| 格式 | JPG / PNG / WebP | 前端校验 + API 端校验 |
| 免费调用量 | 50次/月 | Remove.bg 免费版额度 |
| Worker 请求体 | ≤ 1MB（免费版）| 实际图片走 FormData 不受限，但配额需关注 |
| 并发 | 无限制 | Cloudflare 自动扩缩 |
